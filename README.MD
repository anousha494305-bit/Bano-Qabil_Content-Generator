# 🤖 AI Content Generator using Gemini API

An AI-powered content generation web application built with **Python**, **Streamlit**, and **Google Gemini API**. This application can generate blogs, social media captions, LinkedIn posts, product descriptions, and more.

---

## 🚀 Features

- 📝 Blog Post Generator
- 📷 Instagram Caption Generator
- 💼 LinkedIn Post Generator
- 📘 Facebook Post Generator
- 🛍️ Product Description Generator
- 🎥 YouTube Video Idea Generator
- 📥 Download Generated Content
- ⚡ Fast response using Google Gemini AI
- 🌐 Interactive Streamlit Interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- Google Generative AI SDK

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/anousha494305-bit/Content_Generator.git
```

### 2. Move into Project Directory

```bash
cd Content_Generator
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure Gemini API Key

Get your API key from:

https://aistudio.google.com/app/apikey

Inside `app.py`, replace:

```python
API_KEY = "YOUR_API_KEY_HERE"
```

with your own Gemini API key.

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open in your browser automatically.
